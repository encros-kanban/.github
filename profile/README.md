# Encros

Encros is a Kanban tool designed for agile development teams. It helps teams manage boards, lanes, and cards efficiently. 

⚠️ Note: This repository currently does not contain code. The original project contained sensitive information and is being cleaned before being uploaded. Future updates will include the full code along with additional features.

# Features

- Board management: create, archive, modify status
- Lane (swimlane) management: create, drag & drop, archive
- Card management: create, drag & drop, assign, archive

# Upcoming Features / Improvements

- Real-time collaboration support
- Customizable card templates and labels
